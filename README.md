# Twitter/X Bot

Meme Bot for X/Twitter

Bot ini akan membalas mention secara otomatis:
- Mengambil teks postingan asli
- Mengubah semua huruf vokal menjadi 'i'
- Mengacak kapitalisasi huruf lain
- Queue, backup, dan log otomatis
- Siap deploy ke Heroku & GitHub
